Vue-CLI 는 Vue 프로젝트 생성을 위한 스캐폴딩이다. CLI 는 command line interface의 약자. 작업 순서 step 1. mkdir webpack-test # 폴더 만들기 step 2. cd webpack-test # 폴더 이동 step 3. npm install vue-cli # Vue-CLI 다운로드 step 4. npx vue init webpack # Vue 프로젝트 생성 step 5. npm install # Vue package.json 의 라이브러리 다운로드 step 6. npm run build # 웹팩 빌드. /dist/ 에 합쳐진 파일 만들어짐. /build/build.js 들 사용 step 7. npm run dev # Vue 프로젝트 실행 CLI 템플릿 . webpack-simple Vue 로더 + 빠른 프로토타이핑을 위한 스캐폴딩 웹팩 >> npx vue init webpack-simple . webpack Vue 로더, 핫로더, 린팅테스트, CSS 추출 등 모든 기능을 가진 스캐폴딩 웹팩 >> npx vue init webpack . pwa : 웹팩 기반의 PWA 스캐폴딩 웹팩 >> npx vue init pwa Vue에서 템플릿 파일을 만드는 방법. <template></template> 태그를 사용하는 방법. 파일로 분리 파일 확장자 .vue 해야함. SFC 파일 구조. <template>     <div>         템플릿예제     </div> </template> <style lang="scss" scoped> </style> CLI 스캐폴딩 폴더 구조 : 생성된 프로젝트의 src 폴더 밑에 모듈별로 가독성 높은 폴더를 만들어 사용하는 것이 좋다. ├─ index.html ├─ package.json ├─ README.md ├─ webpack.config.js ├─ build # 빌드 절차 스크립트 ├─ dist # 빌드 결과물 폴더 ├─ node_modules # 모듈 폴더 ├─ config ├─ static ├─ test └─ src ├─ main.js ├─ App.vue ├─ components 컴포넌트 ├─ routes 라우터 ├─ views 라우터 페이지 ├─ store 상태 관리 ├─ api api 함수 ├─ utils 필터 등의 유틸리티 함수 ├─ mixins 믹스인 ├─ plugins 플러그인 ├─ translations 다국어 ├─ images 이미지 ├─ fonts 폰트 └─ assets 기타 자원 └─ logo.png [참고] 페이지 126, 115~125, 184~192, 224~225
