양방향 데이터 바인딩을 실습해 본다.

     이 예제는 하나의 모델 객체를 여러 개의 Vue 객체에서 참조한다.

     app에 바인딩하는 ViewModel은 앙방향 데이터 바인딩하여
     사용자가 입력한 값을 ViewModel 객체를 통해 model 객체의 fruts배열 값을 즉시 변경한다.

     fruits배열에 사용자가 선택한 과일을 추가한 뒤,
     그 결과를 result 안에 있는 span 태그에 바인딩 되어 나타나게 된다.
